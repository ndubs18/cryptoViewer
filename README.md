## CryptoViewer

### SETUP

1. clone repo
2. npm install for /client and /server

   ```console
   cd ./client && npm i && cd ../server && npm i
   ```

3. include provided env file in server/config.env file

4. start server: in /server run

   ```console
       npm start
   ```

5. start client (separate console) in /client run:

   ```console
       npm start
   ```
